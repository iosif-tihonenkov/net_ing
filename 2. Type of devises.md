# Типы устройств, испольуемых в сетях
1. Хаб - он же концентратор. Железка без мозгов, когда на один порт приходит информация - он пересылает эту информацию на все остальные порта. Мозгов нет. OSI: 1
2. Коммутатор - он же свитч, switch. Железка с мозгами. Рассылает информацию согласно своей MAC-таблице адресов - только нужному получателю. Работает внутри AS. OSI: 2
3. Маршрутизатор - он же роутер, router. Рассылает информацию согласно своей IP-таблице адресов. Работает между AS. OSI: 3. Может работать как DHCP и DNS сервер, может иметь различне дополнительные настройки, например - безопасность.
