# Модель OSI 
1. Физический уровень (Physical):
Сигналы: физические (ток, свет, радио), идет кодирование единиц и нулей, модуляция, передача.
Стандарт: Ethernet
Устройства: концентраторы и репитеры - работают с физ сигналов, не декодируют его.

2. Канальный уровень (data Link):
Сигналы: физический сигнал декодируется, ошибки проверяются и исправляются. 
Единица данных: фрейм/кадр
Идентификатор: MAC-адрес
Стандарт: Ethernet
Устройства: Коммутаторы и мосты
Подуровень 1: управление логическим каналом (LLC, Logical Link Control)
Подуровень 2: управление доступом к среде (MAC, Media Access Control).

3. Сетевой уровень (network):
Единица данных: пакет
Идентификатор: IP-адрес
Технологии: протокол ARP 
Устройства: маршрутизатор

4. Транспортный уровень (transport):
Обеспечивает передачу данных по сети
Технологии: протоколы TCP (Transmission Control Protocol) и UDP (User Datagram Protocol) 

5. Сеансовый уровень (session):
Сеансовый уровень управляет соединениями — сессиями: устанавливает их и разрывает. 

6. Уровень представления (presentation)
На шестом уровне происходит преобразование форматов сообщений: кодирование и сжатие. 

7. Уровень приложения (application)
На седьмом уровне, на самой верхушке стека, находятся сетевые службы для конечных пользователей.HTTPS, HTTP, FTP и SMTP.
