### Инструкция по настройке интерфейсов коммутатора Cisco

#### 1. Основные команды для настройки интерфейса

Для настройки параметров интерфейса нужно перейти:
1. Из пользовательского режима — в **режим глобальной конфигурации**:
   ```
   enable
   configure terminal
   ```
2. Затем — в **режим конфигурации интерфейса**:
   ```
   interface <тип_интерфейса> <номер>
   ```
   Пример: `interface FastEthernet 0/1`.

#### 2. Настройка ключевых параметров

После перехода в режим конфигурации интерфейса используйте следующие команды:

* **Скорость порта** (`speed`):
  ```
  speed {auto | 10 | 100 | 1000}
  ```
  * `auto` — автосогласование (по умолчанию);
  * `10`, `100`, `1000` — ручная установка скорости (в Мбит/с).

* **Дуплекс** (`duplex`):
  ```
  duplex {auto | full | half}
  ```
  * `auto` — автосогласование;
  * `full` — полнодуплексный режим;
  * `half` — полудуплексный режим.

* **Текстовое описание** (`description`):
  ```
  description <текстовое_описание>
  ```
  Пример: `description Connection to Server Room`.

#### 3. Пример настройки одного интерфейса

Настройка интерфейса `Fa0/1` со скоростью 100 Мбит/с, полнодуплексным режимом и описанием:
```
enable
configure terminal
interface FastEthernet 0/1
speed 100
duplex full
description Connection to Server Room
```

#### 4. Одновременная настройка нескольких интерфейсов

Используйте команду `interface range` для применения одинаковых настроек к группе портов:
```
interface range <тип_интерфейса> <диапазон>
```
Пример: настройка портов `Fa0/11`–`Fa0/20`:
```
enable
configure terminal
interface range FastEthernet 0/11-20
speed 100
duplex full
description Uplink Ports
```

#### 5. Проверка настроек

* **Команда `show interfaces status`** — отображает статус и настройки всех интерфейсов:
  * описание (`description`);
  * скорость (`speed`);
  * режим дуплекса (`duplex`);
  * статус подключения (`connected` / `notconnect`).

  Пример вывода:
  ```
  Port      Name               Status       Vlan  Duplex  Speed Type
  Fa0/1   Connection to...   notconnect   1     full    100   10/100BaseTX
  Fa0/2                      notconnect   1     auto    auto  10/100BaseTX
  ```

* **Команда `show running-config`** — показывает текущую конфигурацию, включая настройки интерфейсов.
